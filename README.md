# zadania kosolowe nr1 w Python i C#

W tym repo zawarte są rozwiązane problemy programistyczne (prosty kalkulator, konwerter temperatur oraz średnia ocen ucznia) wszystkie konsolowe oraz z menu.

Zadania można odszukac wchodząc w odpowiednią ścieżkę (python lub C#).

### Uruchomienie

1. Wchodzimy w daną ścieżkę oraz plik .py/.cs
2. Kopiujemy kod bądź pobieramy plik, następnie zawartość plików wklejamy do kompilatorów online;
   - python -> https://www.online-python.com/
   - c# -> https://dotnetfiddle.net/
3. Wklejamy kod i uruchamiamy w kompilatorze.
